Building the Fingerspelling Rigs
Summary
Fingerspelling gestures are a set of interactions that allow a user to input numbers from 0 to 9 and a using various combinations of finger taps using a single hand. I built two prototypes to simulate this functionality on a computer mouse and a trackpad. I also coded custom software to onboard users and test their performance in using the prototypes.
